Syntax

Expresions

Literals

  • Booleans true or false
  • Numbers -> JSON definition. If the result has not '.', 'e' or 'E' then is an number-int. Otherwise is a number-float
  • Strings -> JSON definition. (UTF-8 encoded)

Symbols

  • SIGIL + key
  • key

"Key" is any sequence of symbols that:

  • Does not start with a SIGIL
  • Does not have blank, '(' nor ')'
  • Is not a literal

"Blank" is any char value <= ' '

SIGIL's are:

=
Used to normal assignation
!
Used to forced assignation

keys

Values: C char * (letter + letter or digits). They are references to seq's.
Posible presentations:
value
Excecute if exists, or error
(value)
Put value in stack
=value
Assign last stack value
!value
Assign last stack value overwriten

block

A block starts with '(' and ends with ')' (coontrol symbol ')' in comments and strings

Exceptions

A exception stops the program printing a message with the following data:
Message Seq failed (30 characters | 20 characters) Stack -> Name of current blocks in blocks stack
