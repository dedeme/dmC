Exec

exec
Excecute stack 0 (seq) as system command and decards the result
exec=
Excecute stack 1 (seq) as system command and save ouput (string) in stack 0 (seq with one key)
exec=error
Excecute stack 2 (seq) as system command and save error(string) in stack 1 (seq with one key) and ouput (string) in stack 0 (seq with one key)
exec=pipe
Excecute stack 1 (seq) as system command and excecute stack 0 (seq) as system command piping the output of 1 to 0.
