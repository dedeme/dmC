Types

bool

Values: 'true' or 'false'

number-int

Values: C int. Literal without point

number-float

Values: C double. Literal with point

string

Values: C char * (Bitween quotes). Its elements can be accesed by commands rune and char.

Values: List of types.

stacks

Execution stack

  • Contains values
  • Its values are used by commands

Bloks stack

  • Contains seq's in execution. Its last element is the intial program.
  • They have a name. The initial program has the name '-main-'

block

A block starts with '(' and ends with ')' (coontrol symbol ')' in comments and strings

Exceptions

A exception stops the program printing a message with the following data:
Message Seq failed (30 characters | 20 characters) Stack -> Name of current blocks in blocks stack

Commands

keys

Values: C char * (letter + letter or digits). They are references to seq's.
Posible presentations:
value
Excecute if exists, or error
=value
Assign last stack value
!value
Assign last stack value overwriten

External commands

exec
Excecute stack 0 (seq) as system command and decards the result
exec=
Excecute stack 1 (seq) as system command and save ouput (string) in stack 0 (seq with one key)
exec=error
Excecute stack 2 (seq) as system command and save error(string) in stack 1 (seq with one key) and ouput (string) in stack 0 (seq with one key)
exec=pipe
Excecute stack 1 (seq) as system command and excecute stack 0 (seq) as system command piping the output of 1 to 0.
